# MemeHub

MemeHub is a web-based meme generator that allows users to discover meme templates, upload custom images, create memes using draggable text layers, and download their creations instantly.

## Features

### Browse Meme Templates

* Fetches meme templates from the Imgflip API.
* Search templates in real time.

### Upload Custom Images

* Upload personal images and create custom memes.

### Meme Editor

* Add multiple text layers.
* Drag and position text anywhere on the image.
* Edit text content dynamically.
* Customize font size.
* Change text color.
* Change outline color.
* Select different font families.
* Delete selected text layers.

### Download Memes

* Export memes as PNG images with a single click.

### Responsive Design

* Optimized for desktop, tablet, and mobile devices.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Canvas API
* Imgflip API

## How to Use

1. Search for a meme template or upload an image.
2. Open the Meme Editor.
3. Add text layers.
4. Customize text appearance and position.
5. Download the completed meme.
